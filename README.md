#  One-Page Portfolio  
 
Sometimes, we don't have much time to go through an entire portfolio. The best solution? **A sleek, one-page portfolio** that showcases everything in a **clean, modern, and interactive** way. This portfolio is designed for **quick access** to my skills, projects, achievements, and contact details—all in a single scrollable page!  

## Features  
-  **Minimal & Modern Design** – Clean, aesthetic, and user-friendly  
-  **Fully Responsive** – Looks great on all screen sizes  
-  **Smooth Animations** – Engaging hover and scroll effects
